# UP Garage

Sistema web desenvolvido em PHP + MySQL.

## Tecnologias
- PHP
- MySQL
- Bootstrap
- Apache

## Como executar
1. Importar banco:
   banco/up_garage.sql

2. Iniciar Apache e MySQL no XAMPP

3. Acessar:
   http://upgarage.local:8080
   
4. Login Admin do site: 
   admin@upgarage.com
   Senha: 123456
   link:  http://upgarage.local:8080/admin/login.php
